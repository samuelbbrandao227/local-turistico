# Local Turístico

Uma landing page estática em português sobre Busan, na Coreia do Sul, criada para apresentar destinos turísticos, cultura local e experiências de viagem de forma visual e inspiradora.

## Sobre o projeto

Este projeto foi desenvolvido como uma página institucional de viagem, com foco em storytelling e navegação visual. A ideia é apresentar Busan como um destino turístico atrativo, destacando sua mistura de história, arquitetura, natureza e gastronomia.

A interface foi construída com HTML5 e CSS3, utilizando uma estrutura semântica para organizar o conteúdo em seções como:

- introdução à viagem;
- apresentação da cidade;
- lista de atrações imperdíveis;
- rodapé com mensagem final.

## Objetivo

O projeto tem como objetivo demonstrar como criar uma página estática elegante e bem organizada, com foco em:

- layout responsivo;
- tipografia atrativa;
- uso de imagens para reforçar a narrativa;
- experiência visual limpa e moderna.

## Tecnologias utilizadas

- HTML5

## Como visualizar localmente

Você pode abrir o arquivo `index.html` diretamente no navegador, ou rodar um servidor simples para visualização local:

```bash
python3 -m http.server 8000
```

Depois, acesse:

```text
http://localhost:8000
```
